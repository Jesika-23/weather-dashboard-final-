# Weather Dashboard

A responsive Weather Dashboard built using HTML, CSS, and JavaScript that allows users to search for any city and view real-time weather information through API integration.

## Features

- Search weather by city name
- Displays real-time temperature
- Shows humidity and weather condition
- Clean and responsive interface
- Fast and user-friendly design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- OpenWeather API

## How It Works

- Enter the city name
- Click the search button
- Weather data is fetched from the API
- Results are displayed instantly on the dashboard

## Project Purpose

This project was built to strengthen practical knowledge in:

- API integration using JavaScript
- DOM manipulation
- Event handling
- Frontend UI development
